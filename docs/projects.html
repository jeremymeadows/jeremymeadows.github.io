RISC-16 Computer

Starting out with an interest in how computers work at a hardware level, and following a great experience of learning through experimentaton with the Nand Game, I was inspired to see if I could build a programmable computer from scratch. I found a published ISA for a 16-bit RISC processor, so I designed a computer which can execute programs written in that flavor of assembly. After writing the code in an external editor and using a compiler to reduce it down to hexadecimal, copy/paste the data into the EPROM labeled `RISC ASM` and it should run.
